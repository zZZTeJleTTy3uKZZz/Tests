# Styles and UI Kits

## UI kits

Pencil includes pre-built design kits:

  * **Shadcn UI** \- Popular React component library
  * **Halo** \- Modern design system
  * **Lunaris** \- Design system
  * **Nitro** \- Design system

These provide ready-to-use components for common UI patterns.
